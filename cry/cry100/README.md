# cry100

## Description
We have intercepted a file that was being sent to an outside company. We believe there is a message hidden within it can you find it?

http://3.238.30.178:3016

## Solve
This site just has a directory listing with one file `FileFormats.pdf`. The PDF is talking about file formats so maybe there is a hidden file within the PDF.

`binwalk` is a tool that will look for embedded files. Running `binwalk -e FileFormats.pdf` detected an embedded zip file with a text file in it `msg.txt`.

A quick google search for "1981 arcade game urban legend" brings up results for Polybius. However, Polybius is also the name of a substitution cipher that uses a grid of letters to encrypt and decrypt text.

A-Y tells us we are using the letters A through Y in our grid (25 letters since it must be square), and 5x5 is the grid size.

The ciphertext we must decrypt is `311555:3324454311`

This site https://www.dcode.fr/polybius-cipher can help us with this. I pasted `311555:3324454311` into the ciphertext field, set the grid to 5x5 and typed `ABCDEFGHIJKLMNOPQRSTUVWXY` as the key.

Pressing "Decrypt" gives us the first part of the challenge:
```
KEYMITRA
```

The second half of the challenge gives us a clue "series of interwoven ciphers" and some text to decode `YKT{r98c829m5j05xuf5m67mu78101vd8m77m1v0369r`. Searching "series of interwoven ciphers" brings up results for a Vigenere cipher, which uses interwoven ceasar ciphers and a key to encrypt/decrypt text.

I used https://www.dcode.fr/vigenere-cipher to solve this. `YKT{r98c829m5j05xuf5m67mu78101vd8m77m1v0369r}` went in the ciphertext input, the alphabet is just regular (A-Z), and the key is `MITRA` which we found in the previous step.
